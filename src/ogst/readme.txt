LaTeX2e v2.0 class for OGST

What's New in OGST v2.0 (June 2011)

- The 2011 update is now compatible with the recent version of LaTeX and standard installations.

- The commands are more user friendly.  There is no need to install special fonts. 
Please note that in the last stage, the publisher may change some fonts to perform the page layout of the journal.

The following files are part of the macro package for OGST:

- readme.txt: Information (equivalent to this page)
- ifpen-ogst.cls: The document class file 
- ifpen-ogst-doc-en.pdf: User's Guide (english)
- ifpen-ogst-doc-fr.pdf: User's Guide (french)
- template.tex: Example of an article (LaTeX source)
- packages.zip: Necessary for the old LaTeX versions (see before the user's guide).

All files are compressed in a single archive:
ifpen-ogst-package.tar.gz
or
ifpen-ogst-package.zip

Please contact your system administrator to install it.

Les nouveautés de la classe OGST, v2.0 (Juin 2011)
 
- La version 2011 est compatible avec les installations récentes et standardisées de LaTeX.

- Les commandes sont beaucoup plus simples. Il n'est pas nécessaire d'installer d'autres fontes. 
Celles-ci seront remplacées au final par le compositeur.

Les macros contiennent les fichiers suivants :

- readme.txt: cette page d'informations
- ifpen-ogst.cls: la classe principale 
- ifpen-ogst-doc-en.pdf: le guide de l'utilisateur (anglais)
- ifpen-ogst-doc-fr.pdf: le guide de l'utilisateur (français)
- template.tex: un exemple de document (source LaTeX )
- packages.zip: il faut installer ces styles si vous avez une ancienne version de LaTeX (consulter d'abord le guide).

Tous les fichiers sont compressés dans une archive unique :
ifpen-ogst-package.zip
ou
ifpen-ogst-package.tar.gz

Contacter votre administrateur système pour installer ce macro package.
